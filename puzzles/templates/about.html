{% translate "什么是Puzzle Hunt？" %}

{% blocktranslate %}你可能已经接触过各种各样的谜题：杂志上的数独和爱因斯坦逻辑谜题，语文课上的灯谜，英语课上的填字游戏。也许你和同学玩过你画我猜，曾用密码加密信息让朋友破译；看过象棋或斗地主的残局问题，读过那些引人入胜的推理小说。或者，你的 Steam 库里有《Baba is You》这样的推箱子游戏，又或者在闲暇时光去过密室逃脱。{% endblocktranslate %}

{% blocktranslate %}在Puzzle Hunt（解谜寻宝）中，你和你的团队需要解决的不止是一道，而是一系列这样的谜题，并从中发掘隐藏的线索。{% endblocktranslate %}

{% blocktranslate %}想象一下，任何事物都可以成为谜题：看似是填字游戏，实际上是对填字格的推理；或是色块的组合，每组方块代表一个动漫角色。神奇的谜语，严密的逻辑，隐藏的图片，甚至可能是一首歌曲中歌手演唱的顺序，或是一部动画片里第一集结尾的密码，万事万物皆可成谜。{% endblocktranslate %}

{% blocktranslate %}而你的目标，就是找到合适的线索，选择最符合当前情况的方法，从里面找到你的答案。无论如何，最终你一定能得到一组有意义的词语或一句话。{% endblocktranslate %}

{% blocktranslate %}在解决一连串的谜题之外，元谜题（Meta Puzzle）会将你之前得到的所有答案串联成新的谜题，形成错综复杂的结构。{% endblocktranslate %}

{% blocktranslate %}最终，Puzzle Hunt 的目标是团队共同解决所有出现的元谜题（是的！你不需要所有小题的答案就可以解决元谜题！），得到隐藏在谜题和谜题结构中的最终答案（通常被称为「Final Meta」）。{% endblocktranslate %}
